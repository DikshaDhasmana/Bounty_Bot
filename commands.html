Bot Commands

The bot has a number of commands including utility, moderation, gaming and shop commands. Given below are the lists of commands with their descriptions.

Utility Commands:

Command Description
.daily Claim your daily rewards.
.pro or .pro @ View your/someone's server profile.
.si View server information.
.lb View server leaderboard.
.rank or .rank @ View your/someone's server rank.
.bounty or .bounty @ View your/someone's Bounty Poster.
.av or .av @ View your/someone's avatar.
.com View this message.

Nakama Commands:

Command Description
.an @ Add someone to your Nakama list.
.ra @ Remove someone from your Nakama list.
.db @ <amount> Donate Berry to any of your Nakama.

Fun Commands:

Command Description
.meme Get an Anime meme.
.op_meme Get an One Piece meme.
.roast @ Roast someone. (Note: Might be offensive).
.roast_hindi @ Roast someone in Hindi. (Note: Might be abusive).
.sd @ Send anonymous DM to someone. (Note: Kindly report to the server if received any unwanted offensive DM.)

Action Commands:

Command Description
.hug @ Hug someone.
.kiss @ Kiss someone.
.smooch @ Give a smooch to someone.
.slap @ Slap someone.
.punch @ Punch someone.
.kill @ Kill someone.
.mock @ Make fun of someone.
.laugh Laugh.
.sad Show sad emotion.
.happy Show happy emotion.

Moderation Commands:

Command Description
.kick @ Kick someone from server.
.ban @ Ban someone from server.
.unban <id> Unban someone from server.
